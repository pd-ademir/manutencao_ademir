Como Funciona?

Este painel permite cadastrar novos itens ou atualizar os existentes a partir de um arquivo CSV. O sistema utiliza uma coluna-chave para cada tipo de dado:

  • Placas: A coluna placa é a chave.
  • Motoristas: A coluna cpf é a chave.
  • Conjuntos: A coluna nome_conjunto é a chave.

Lógica de Atualização: Se um item com a mesma chave já existir, o sistema compara todas as outras colunas. Se houver qualquer diferença, ele atualiza o registro. Se não houver diferenças, ele ignora a linha. Itens com chaves que não existem no banco de dados serão criados.

1. Upload de Placas

Use o modelo para cadastrar/atualizar placas. Campos de data devem usar o formato YYYY-MM-DD.

Use o modelo para cadastrar/atualizar motoristas. A senha será os 6 primeiros dígitos do CPF.

Use o modelo para cadastrar/atualizar conjuntos. As placas e o CPF do motorista devem existir.
